Privacy Policy

Your privacy is our priority. Here's how we protect it.

Data Collection & Storage

  • No Tracking: We don't collect personal data or use analytics.
  • Local Storage Only: All data remains on your device. We use Chrome's local storage for session management, but this data is encrypted and temporary.
  • Session Security: Your master password is temporarily stored in memory and local storage (encrypted) for session management. Sessions automatically expire after 12 hours for security.

Encryption & Security

  • Local Encryption: All data is encrypted on your device using your master password. We never see or store it.
  • Encrypted Data Only: When you save, we receive only encrypted content â€” never plaintext.
  • Chunked Encryption: Large data is automatically split into smaller chunks for secure processing, ensuring stable encryption without memory issues.
  • Encryption Control: You choose the number of encryption rounds. One round is usually enough, but more can add extra protection.
  • Security Enforcement: Saving is blocked if encryption rounds are set to zero (plaintext). We enforce encryption to protect your data.

User Control & Transparency

  • User-Controlled Output: You control how your data is encrypted and how it looks after encryption.
  • Session Management: You can manually log out at any time to clear your session. Sessions automatically expire for security.
  • Data Recovery: If a session expires, any unsaved data remains encrypted and secure.
  • Open Source: Our code is public and auditable for transparency.

Firebase Usage

  • Authentication: We use Firebase Authentication for secure user management. Only your email is stored by Firebase for account purposes.
  • Data Storage: Encrypted data is stored in Firebase Firestore. Firebase never receives or stores your master password or unencrypted data.
  • Firebase Privacy: Our Firebase implementation follows Google's privacy standards. You can review Firebase's privacy policy at https://firebase.google.com/support/privacy

Technical Details

  • Local Storage: We use Chrome's storage API for session management. This data is encrypted and automatically cleared on session expiration.
  • Memory Management: Large data is processed in chunks to ensure stable performance and security.
  • Automatic Cleanup: All sensitive data is automatically cleared from memory when you log out or when your session expires.
